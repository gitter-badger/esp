Installing on MAC OS X

The Embedthis ESP install distribution for Mac OSX is published as a an installable pkg file.

Installing the PKG Image

  1. Double click on the PKG image.

Removing ESP

  1. Run a terminal and change directory to the installation directory for ESP. This is typically "/etc/appweb".

  2. Run the remove script in the install directory for ESP by typing:

    sudo ./uninstall
